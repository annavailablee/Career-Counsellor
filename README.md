## Overview

CSE Career Compass helps students identify the tech career path that aligns with their skills and interests.

After answering around 20–25 questions, users receive:
﹂ Most ideal career match
﹂ Top 8 recommended career options
﹂ Skill breakdown across major CS domains such as Web Development, Cybersecurity, AI/ML, Data Science, Cloud, Mobile Development, and more

---

## Features

﹂ Simple and responsive UI  
﹂ Personalised career recommendations  
﹂ Domain-wise skill visualisation  
﹂ Expandable structure for future enhancements

## Installation & Setup

Requires Node.js (v14+) and npm installed

```bash
git clone https://github.com/annavailablee/CS-Career-Compass.git
cd CS-Career-Compass
npm install
npm start
```
